# Nodejs-expressjs

Plantilla para tercer parcial potenciado con nodejs y express

## Herramientas importantes

- [expressgenerator](https://expressjs.com/fr/starter/generator.html) v4.16

### Dependencias adicionadas (not included in express generator)

- [nodemon](https://nodemon.io/) - v1.19

# Instalacion

1. Clonar el proyecto desde https://github.com/csullaez/examen-plantilla.git
2. Instalar dependencias con
   `npm install`
3. Desplegar proyecto en ambiente desarrollo
   ` npm run dev`
